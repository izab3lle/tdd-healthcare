# Sobre o projeto
HealthCare é um projeto desenvolvido com TDD para a cadeira de Gerência de Configuração e Testes. Possui a implementação em Java do módulo de cadastro de um sitema simples de hospital.

**Tecnologias utilizadas**:
- JUnit;
- Mockito;
- Spring Boot;
